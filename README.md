
Alien Invasion

Alien Invasion is a 2D space shooter that I built by following along with the book "Python Crash Course" by Eric Matthes.
Working on this game introduced me to pygame fundamentals including sprites, key/mouse events, rendering fonts, 
screen display,object collisions, and dynamic HUD elements (score, lives, and level display).

Dependencies:
pygame
python3

Run the game via the command line with "python3 alien_invasion.py"
